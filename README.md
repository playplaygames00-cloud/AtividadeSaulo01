O objetivo deste espaço é documentar a resolução de problemas lógicos e o aprendizado da linguagem **Python**. Como estou vindo de uma base em **C#**, este repositório também serve para registrar a transição de sintaxe e o domínio das particularidades do Python (como a ausência de chaves e o uso da indentação).
