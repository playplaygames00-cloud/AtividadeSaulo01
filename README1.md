# Atividade Aula 01: Calculadora de IMC ⚖️

Este projeto marca o início dos meus estudos em Python no Google Colab.

## 📝 Descrição
O script solicita ao usuário sua **altura** e seu **peso**, realiza o cálculo do Índice de Massa Corporal (IMC) e retorna a classificação de saúde baseada nos padrões médicos.

## 🛠️ Conceitos Aplicados
* **Entrada de Dados:** Uso do `input()` com conversão para `float`.
* **Manipulação de Output:** Uso do `clear_output` da biblioteca `IPython` para limpar a tela no Colab.
* **Estruturas Condicionais:** Implementação de lógica com `if`, `elif` e `else`.
* **Formatação:** Uso de `f-strings` e a função `round()` para limitar as casas decimais.

## 🚀 Como rodar
1. Abra o arquivo `Atividade_Aula_01.ipynb` no Google Colab.
2. Execute a célula de código.
3. Insira os dados solicitados no console que aparecerá abaixo da célula.
